Thoughts on Progresive Enhancement

It is the process where you take content and slowly add layers via css then javascript. This allows others to obtain what is can't be seen without javascript. It works well because instead of start with a complicated mess you start with a simple task. After you make the simple task you add more to it so it can look nice. After you make it look nice then you add on extra features that are nice but it can do without. If it can't use the features then it goes to the original presentation.
